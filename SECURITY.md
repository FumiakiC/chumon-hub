# Security Policy

## Supported Versions

現在、セキュリティアップデートが提供されているバージョンは以下の通りです。

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

※ 現在は開発段階（0.x.x）のため、最新のコミット（mainブランチ）のみをサポート対象とします。

## Reporting a Vulnerability

脆弱性を発見された場合、GitHubのIssueには**絶対に書き込まないでください**。公開された場で脆弱性を報告することは、悪意のある攻撃者にヒントを与えることになります。

脆弱性の報告は、以下のメールアドレスまで直接ご連絡ください。

**Email:** small.board7213@fastmail.com

報告には以下の情報を含めていただけると幸いです。

* 脆弱性の種類と影響範囲
* 再現手順（PoCコードやスクリーンショット）

報告を受け取り次第、速やかに内容を確認し、対応方針をご連絡いたします。
報告者の方には、問題が解決され一般に公開されるまでの間、情報の機密性を保持していただくようお願いいたします。

## Security Measures

本プロジェクトでは、以下のセキュリティ対策を実施しています。

* **No Secrets in Code:** APIキーや認証トークンはコードベースに含まれず、環境変数として注入されます。
* **Zero Trust:** 本番環境へのアクセスは Cloudflare Zero Trust により保護されており、JWT署名検証を行っています。
* **Stateless:** アップロードされたデータは一時的な処理にのみ使用され、サーバー内に永続保存されません。
